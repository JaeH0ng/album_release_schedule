# 이번 주 계획

## 주간 정보

- 기간: 2026-06-18 ~ 2026-06-21
- 이번 주에 이미 잡힌 다른 일: 미입력
- 앨범에 현실적으로 사용할 수 있는 총시간: 우선 2시간 확보 목표, 실제 가능 시간 확인 필요

## 먼저 잠글 작업 시간

| 구분 | 날짜/시간 | 장소 | 작업 | 최소 결과물 |
| --- | --- | --- | --- | --- |
| 메인 블록 | 2026-06-18 ~ 2026-06-20 중 60분 | 아티스트 지정 | 녹음 템플릿과 입력 레벨 준비 | 저장 경로, 파일명, 기타/보컬 입력 확인 |
| 메인 블록 | 2026-06-21 중 60분 | 아티스트 지정 | `good night` 판단용 데모 | 전체 1테이크, 임시 키/BPM, 구조 메모 |
| 예비 블록 | 2026-06-21 중 30분 | 아티스트 지정 | `good night` 최소 데모 | 멈추지 않은 전체 1테이크와 임시 BPM 1개 |

## 이번 주 결과물

- 반드시 끝낼 것 1개: 2026-06-21까지 `good night` 판단용 데모 1개
- 가능하면 끝낼 것 1개: BPM 후보 2개를 비교하고 더 편한 쪽 기록
- 이번 주에는 하지 않을 것: 편곡 확정, 플러그인 비교, 음색 다듬기, 세 번째 데모 테이크

## 시간이 무너졌을 때

- 90분 가능: 60분 세션을 마친 뒤 최대 30분 동안 청취 메모와 파일 정리
- 30분 가능: 최고음 구간으로 키 확인 5분, BPM 선택 5분, 전체 1테이크 15분, 파일명과 메모 5분
- 10분만 가능: 녹음 경로와 입력을 확인하고 다음 30분 세션을 바로 시작할 수 있게 템플릿 저장
