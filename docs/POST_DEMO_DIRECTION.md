# 데모 이후 방향 논의

이 문서는 `2026-07-21` 전곡 판단용 데모 마감 직후, `2026-07-22`부터 `2026-08-01` 사이에 다음 단계를 정리하기 위한 회의 메모 기준본이다.

데모 단계에서는 통기타+보컬 중심이지만, 여기서는 "간단한 편곡까지 이미 한번 얹어본 상태"라고 가정하고 다음 결정을 내린다. 임시 발매일 `2026-12-04`, 유통사 전달 마감 `2026-11-13`은 유지한 채 편곡과 본녹음으로 넘어가는 것이 목표다.

## 먼저 정리할 것

- 전곡 중 지금 바로 본녹음 후보로 밀 수 있는 곡
- 데모는 괜찮지만 편곡 방향이 아직 애매한 곡
- 키/BPM을 다시 만질 최대 4곡
- 최종 수록 후보 8~9곡의 1차 윤곽

## 곡별로 의논하면 좋은 질문

- 이 곡은 데모만으로도 중심 에너지가 살아 있는가
- 간단한 편곡을 얹었을 때 오히려 중심이 흐려지지 않았는가
- 통기타와 보컬만 남겨도 버티는가, 아니면 추가 악기가 사실상 필수인가
- 지금 키와 BPM이 감정, 발음, 호흡에 맞는가
- 구조를 줄이거나 늘려야 할 부분이 있는가
- 후렴, 브릿지, 마지막 반복 중 가장 먼저 손봐야 할 구간은 어디인가
- 이 곡의 "다음 편곡 실험 1순위"는 무엇인가

## 편곡 쪽에서 결정할 것

- 앨범 전체에서 반복될 공통 질감은 무엇인가
- 곡마다 겹치지 않게 남길 차별점은 무엇인가
- 데모에서 이미 먹힌 아이디어와 버릴 아이디어는 무엇인가
- 악기 추가 우선순위는 무엇인가
- 리듬이 먼저인 곡과 공간감이 먼저인 곡은 각각 무엇인가
- 코러스, 패드, 퍼커션, 베이스, 일렉 기타 중 실제로 필요한 파트만 남기면 무엇인가

## 장비와 작업 환경에서 결정할 것

- 모니터링 스피커를 언제까지 구매할지
- 스피커 배치와 최소한의 청취 환경을 언제 셋업할지
- 편곡 판단은 스피커 기준으로 하고, 이어폰/헤드폰은 어떤 용도로만 쓸지
- 편곡 단계에서 꼭 필요한 플러그인, DI, 마이크 운용은 무엇인지

## 결과로 남겨야 할 것

- 최종 수록 후보 8~9곡
- 재테스트가 필요한 최대 4곡
- 곡별 편곡 출발점 1줄
- 앨범 공통 팔레트 메모
- 모니터 스피커 구매/배치 마감일
- `2026-08-23` 편곡 방향 확정 전까지의 주간 실험 순서
