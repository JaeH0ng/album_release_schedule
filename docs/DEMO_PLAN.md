## 데모 한 곡의 완료 조건

- [ ] 처음부터 끝까지 이어지는 통기타+보컬 파일 1개
- [ ] 녹음한 키와 카포 위치 기록
- [ ] 임시 BPM 또는 자유 템포 여부 기록
- [ ] 현재 곡 구조 기록
- [ ] 가장 편한 구간과 가장 불편한 구간 기록
- [ ] 다음 테스트 한 가지 기록
- [ ] 곡 문서에 파일 경로와 메모 연결

연주 실수, 음정 실수와 생활 소음은 키/BPM 판단을 방해하지 않는 한 다시 녹음할 이유가 아니다.

## 시간이 부족한 날의 최소 데모

30분밖에 없다면 다음만 수행한다.

1. 후렴 최고음으로 키를 확인한다.
2. 임시 BPM 하나를 선택한다.
3. 멈추지 않고 전체를 한 번 녹음한다.
4. 파일명에 날짜와 버전을 적는다.

이 최소 데모도 판단이 가능하면 완료로 인정한다.
